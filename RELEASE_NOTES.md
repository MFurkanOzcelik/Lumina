## ğŸš€ How to Use

### Adding Tags
1. Select a note
2. Right-click and choose "Add Tag"
3. Type the tag name (existing tags will be suggested automatically)
4. Press Enter or click on a suggested tag

### Filtering by Tags
1. Click on a tag in the sidebar
2. All notes with that tag will be displayed
3. Click again to remove the filter

### Removing Tags
1. Select a note
2. Right-click and choose "Remove Tag"
3. Select the tag you want to remove
